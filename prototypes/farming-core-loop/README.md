# 农场核心循环原型

## 原型目标

验证农场核心玩法循环的可行性和玩家体验。

## 核心问题

**农场核心循环（耕地→播种→浇水→收获）的操作反馈和时间节奏是否令人满意？**

## 操作指南

### 工具切换
- **[1]** 锄头 - 开垦荒地
- **[2]** 种子 - 播种（番茄，4天成熟）
- **[3]** 浇水壶 - 浇水

### 游戏操作
- **点击地块** - 使用当前工具操作
- **[空格]** - 跳过一天，所有作物生长
- **[R]** - 重置游戏

### 游戏循环
1. 用锄头开垦荒地 → tilled
2. 切换到种子，播种 → planted
3. 切换到浇水壶，浇水 → growing
4. 按空格跳过一天
5. 重复浇水直到成熟 → harvestable
6. 点击收获，获得种子奖励

## 测试要点

- [ ] 操作反馈是否即时？
- [ ] 体力消耗是否合理？
- [ ] 生长周期是否合适？
- [ ] 收获动画是否满足？
- [ ] 整体节奏是否令人上瘾？

## 运行方式

1. 用 Godot 4.6 打开此目录
2. 点击运行（Play）
3. 测试上述操作
