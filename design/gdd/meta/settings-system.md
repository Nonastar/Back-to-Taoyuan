# 偏好设置系统 (Settings System)

> **Status**: Approved
> **Author**: AI-assisted Design
> **Last Updated**: 2026-04-08
> **Implements Pillar**: 所有系统 (贯穿全游戏)

## Overview

偏好设置系统管理游戏的所有可配置选项，包括音量控制、显示设置、按键绑定和游戏偏好。玩家可以随时调整这些设置，系统会将所有偏好保存到存档并实时应用到游戏中，让每个玩家都能获得个性化的游戏体验。

## Player Fantasy

玩家应该感受到"这个游戏尊重我的时间和偏好"。设置界面应该简洁明了，玩家能在10秒内找到并调整任何想要的功能，而不必记住复杂的菜单路径。每次调整都应该立即生效，让玩家感受到对游戏的掌控力。

## Detailed Design

### Core Rules

#### 1. 设置分类

| 分类 | 子项 | 持久化 |
|------|------|--------|
| 音频 | BGM音量、SE音量、语音音量、UI音效 | 是 |
| 显示 | 窗口模式、分辨率、画质、UI缩放 | 是 |
| 游戏 | 自动存档间隔、教程显示、语言 | 是 |
| 控制 | 键位绑定、鼠标灵敏度、触屏布局 | 是 |
| 辅助 | 文字大小、色盲模式、字幕 | 是 |

#### 2. 音量控制

- **BGM音量**: 0-100整数滑块，步进5
- **SE音量**: 0-100整数滑块，步进5
- **语音音量**: 0-100整数滑块，步进5
- **UI音效**: 0-100整数滑块，步进5
- **静音切换**: 一键静音，所有音量归零

#### 3. 键位绑定

- **预设方案**: 键盘默认、键盘 WASD、手柄默认、手柄对称
- **自定义绑定**: 7个主要动作可单独绑定
  - 移动上/下/左/右
  - 确认/取消
  - 背包
- **冲突检测**: 同一按键不能绑定多个动作，冲突时拒绝保存

#### 4. 显示设置

- **窗口模式**: 全屏、窗口、无边框窗口
- **分辨率**: 根据显示器自动推荐，支持自定义
- **画质**: 低/中/高/极高 (影响粒子、特效、LOD)
- **UI缩放**: 80%-150%，步进10%
- **帧率限制**: 30/60/120/无限制

#### 5. 辅助功能

- **文字大小**: 小/中/大/特大
- **色盲模式**: 无/红色盲/绿色盲/蓝黄色盲
- **字幕显示**: 开/关
- **提示速度**: 慢/中/快 (教程文字显示速度)

### States and Transitions

```
┌─────────────────────────────────────────────────────────────┐
│                     设置系统状态机                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [LOADING] ──设置加载完成──> [IDLE]                        │
│       ↑                              │                      │
│   游戏启动                        打开设置面板               │
│       │                              ↓                      │
│  [UNLOADED] <───退出游戏───── [SETTINGS_OPEN]              │
│                                        │                     │
│                               关闭/保存/取消                 │
│                                        ↓                     │
│                                  [IDLE] ──→ [REVERTING]     │
│                                              │               │
│                                        恢复默认设置           │
│                                              ↓               │
│                                         [IDLE]              │
└─────────────────────────────────────────────────────────────┘
```

| State | Entry Condition | Exit Condition | Behavior |
|-------|----------------|----------------|----------|
| UNLOADED | 游戏退出 | 设置数据加载 | 无操作 |
| LOADING | 设置加载中 | 从存档读取完成 | 显示加载指示 |
| IDLE | 游戏运行中 | 用户打开设置 | 监控系统状态 |
| SETTINGS_OPEN | 用户打开设置 | 关闭/保存/取消 | 显示设置面板 |
| REVERTING | 恢复默认中 | 恢复完成 | 执行恢复逻辑 |

### Interactions with Other Systems

#### 与 F05 音效系统
- **输入**: 接收音量参数
- **输出**: 应用音量到所有音效通道
- **接口**: `SetMasterVolume(type, value)`, `SetChannelVolume(channel, value)`

#### 与 F04 存档系统
- **输入**: 读取存档中的设置数据
- **输出**: 保存设置数据到存档
- **接口**: `LoadSettings()`, `SaveSettings()`

#### 与 U01 HUD系统
- **输入**: UI缩放参数
- **输出**: 应用缩放到所有UI元素
- **接口**: `ApplyUIScale(scale_factor)`

#### 与 F02 天气系统 (画质相关)
- **输入**: 画质等级参数
- **输出**: 天气粒子效果等级
- **接口**: `SetWeatherQuality(level)`

#### 与 X01 教程系统
- **输入**: 教程相关偏好
- **输出**: 教程显示速度和频率
- **接口**: `SetTutorialSpeed(level)`, `SetTutorialEnabled(enabled)`

## Formulas

### UI缩放因子计算

```
actual_scale = base_scale + (slider_value / 100) * scale_range
```

| Variable | Type | Range | Source | Description |
|----------|------|-------|--------|-------------|
| base_scale | float | 0.8 | 配置表 | 最小缩放 |
| slider_value | int | 0-100 | 用户输入 | 滑块位置 |
| scale_range | float | 0.7 | 配置表 | 缩放范围 (max - min) |
| actual_scale | float | 0.8-1.5 | 计算结果 | 最终缩放因子 |

**Expected output range**: 0.8-1.5
**Edge case**: slider_value超出0-100范围时，clamp到0或100

### 音量分贝映射

```
db_value = -60 + (slider_value / 100) * 60
```

| Variable | Type | Range | Source | Description |
|----------|------|-------|--------|-------------|
| slider_value | int | 0-100 | 用户输入 | 滑块值 |
| db_value | float | -60 to 0 | 计算结果 | 分贝值 |
| linear_value | float | 0.0-1.0 | 引擎使用 | 线性音量 |

**Expected output range**: -60dB (静音) 到 0dB (最大)
**Edge case**: slider_value = 0 时，db_value = -60dB，即完全静音

### 画质等级效果系数

```
effect_quality = base_quality * (1 + quality_level * 0.25)
```

| Variable | Type | Range | Source | Description |
|----------|------|-------|--------|-------------|
| base_quality | float | 0.5 | 配置表 | 低画质基准 |
| quality_level | int | 0-3 | 用户选择 | 0=低,1=中,2=高,3=极高 |
| effect_quality | float | 0.5-1.25 | 计算结果 | 效果系数 |

**Expected output range**: 0.5-1.25

## Edge Cases

| Scenario | Expected Behavior | Rationale |
|----------|------------------|-----------|
| 设置值超出范围 | 强制clamp到有效范围 | 防止崩溃 |
| 键位冲突 | 拒绝保存，显示错误提示 | 避免操作混乱 |
| 显示器不支持分辨率 | 自动回退到上一个可用分辨率 | 鲁棒性 |
| 存档损坏无法加载设置 | 使用内置默认设置 | 防止游戏无法启动 |
| 音量设置为0但需要播放 | 静默跳过，不报错 | 避免警告刷屏 |
| 语言文件缺失 | 回退到游戏内置语言(中文) | 多语言容错 |
| 全屏切换失败 | 保持当前模式，显示提示 | 避免游戏卡死 |

## Tuning Knobs

| Parameter | Current Value | Safe Range | Effect of Increase | Effect of Decrease |
|-----------|--------------|------------|-------------------|-------------------|
| volume_step | 5 | 1-10 | 滑块更精细 | 滑块更粗糙 |
| ui_scale_min | 0.8 | 0.5-1.0 | 可缩小更多 | 最小缩放更大 |
| ui_scale_max | 1.5 | 1.2-2.0 | 可放大更多 | 最大放大受限 |
| auto_save_interval | 5分钟 | 1-30分钟 | 存档更频繁 | 存档更稀疏 |
| key_bind_timeout | 5秒 | 2-10秒 | 更多时间思考 | 需要更快决定 |

## Acceptance Criteria

- [ ] 音量调整立即生效，无需重启
- [ ] 键位冲突检测阻止非法绑定
- [ ] 设置正确保存到存档并在加载后恢复
- [ ] UI缩放在所有分辨率下正常工作
- [ ] 色盲模式正确应用颜色滤镜
- [ ] 画质切换不导致游戏崩溃
- [ ] 恢复默认设置后所有选项回到初始值
- [ ] 性能: 设置系统内存占用 < 1MB
- [ ] 无硬编码设置值，所有值可配置
