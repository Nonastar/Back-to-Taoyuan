> **Festival**: 春节 (农历新年)

## Overview

包饺子小游戏是春节节日活动中的传统手艺玩法。玩家通过"擀皮+包馅+捏合"三步流程制作饺子，在限定时间内尽可能多地包出美观的饺子。

游戏采用节奏操作模式：
- **擀皮**：控制擀面杖滚动，制作大小均匀的面皮
- **包馅**：将馅料放入面皮中央
- **捏合**：快速捏出褶皱，封口成型

每场游戏有60秒时间，根据包出的饺子数量和美观程度获得积分。

小游戏独立于主游戏运行，完成后根据成绩与活动系统交互进行奖励结算。

## Player Fantasy

包饺子小游戏给玩家带来**春节家庭团聚的温馨感和手艺精进的成就感**。玩家应该感受到：

- **揉面的期待** — 看着面团在手中逐渐成形
- **擀皮的专注** — 控制力度，制作完美的圆形面皮
- **包馅的技巧** — 将馅料精准放入面皮中央
- **捏褶的韵律** — 快速而有节奏地捏出漂亮的褶皱
- **成品的满足** — 看着一排排包好的饺子，满满的成就感
- **春节的氛围** — 一家人围坐一起包饺子的温馨场景

参考游戏：
- **Overcooked**: 节奏厨房的紧张感
- **煮糊了2**: 混乱中的协作乐趣
- **Pizza Tower**: 快速操作的爽快感

**情感曲线**:
1. **开场**: 厨房场景，春节装饰，温馨氛围
2. **前期**: 熟悉操作，节奏逐渐建立
3. **中期**: 速度加快，感受操作的流畅
4. **后期**: 追求完美，挑战高评分饺子
5. **结算**: 查看成果，分享喜悦

**与春节的关联**:
- 春节是团圆的日子，包饺子象征着家庭和睦
- 游戏中的 NPC 可以是家人或邻居，一起参与
- 背景音乐可以是中国传统新年音乐

## Detailed Design

### Core Rules

#### 1. 游戏时间

```
游戏时长: 60 秒
倒计时显示: 屏幕顶部显示剩余时间
最后10秒: 时间显示变红，播放警报音
时间耗尽: 游戏结束，进入结算
```

#### 2. 操作三阶段

##### 2.1 擀皮阶段

```
操作: 玩家使用滚轴擀面皮
目标: 将面团擀成圆形面皮

擀皮判定:
- 大小: 达到目标大小 (60-80% 面积比例)
- 形状: 圆形度 ≥ 80%
- 厚度: 均匀度 ≥ 70%

评分等级:
| 等级 | 条件 | 基础分 |
|------|------|--------|
| 完美 | 大小+形状+厚度全部优秀 | 30分 |
| 良好 | 两项达标 | 20分 |
| 一般 | 一项达标 | 10分 |
| 失败 | 三项都不达标 | 0分 (浪费面团) |

操作技巧:
- 力度均匀，避免面皮厚薄不均
- 每次滚动后旋转面皮，保持圆形
- 擀到边缘时收力，避免面皮过大或过小
```

##### 2.2 包馅阶段

```
操作: 玩家使用勺子取馅，放入面皮中央

取馅量: 系统自动分配固定量

放置判定:
- 位置: 馅料中心在面皮中心 ±20% 半径内
- 量: 馅料不溢出

评分等级:
| 等级 | 条件 | 基础分 |
|------|------|--------|
| 精准 | 位置正中，量合适 | 20分 |
| 良好 | 位置偏离<10%，量合适 | 15分 |
| 一般 | 位置或量略有偏差 | 10分 |
| 溢出 | 馅料溢出面皮边缘 | 5分 |
```

##### 2.3 捏合阶段

```
操作: 玩家快速点击/滑动，捏出饺子褶皱

捏褶要求:
- 总共需要捏 8-12 个褶皱
- 每个褶皱需要在正确位置
- 操作速度影响最终评分

评分等级:
| 等级 | 条件 | 基础分 |
|------|------|--------|
| 完美 | 12个褶皱，位置准确，捏合紧密 | 50分 |
| 良好 | 10个褶皱，位置较准 | 35分 |
| 一般 | 8个褶皱，位置偏差 | 25分 |
| 粗糙 | <8个褶皱 | 15分 |

操作技巧:
- 均匀节奏，不要忽快忽慢
- 点击/滑动位置要准确
- 保持稳定的节奏感
```

#### 3. 单个饺子评分

```
单个饺子总分 = 擀皮分 + 包馅分 + 捏合分

| 评价 | 总分范围 | 称号 |
|------|----------|------|
| 完美 | 90-100分 | ⭐⭐⭐ 完美 |
| 优秀 | 75-89分 | ⭐⭐ 优秀 |
| 合格 | 50-74分 | ⭐ 合格 |
| 不合格 | <50分 | 破皮 |
```

#### 4. 连击系统

```
连击计数: 连续包出"合格"及以上评分的饺子时连击数+1
连击中断: 包出"不合格"饺子时重置为0

连击奖励:
| 连击数 | 额外加分 |
|--------|----------|
| 3连 | +20分 |
| 5连 | +50分 |
| 8连 | +100分 |
| 10连 | +150分 |
```

#### 5. 奖励结算

##### 5.1 分数段奖励

| 分数段 | 称号 | 金币奖励 |
|--------|------|----------|
| ≥2200分 | 面点大师 | 500g |
| ≥1800分 | 包饺高手 | 300g |
| ≥1300分 | 包饺新秀 | 150g |
| ≥600分 | 参与奖 | 50g |

##### 5.2 额外奖励

| 条件 | 额外奖励 |
|------|----------|
| 首次获得"面点大师" | 成就解锁 |
| 完美饺子≥5个 | +100g |
| 刷新个人最高分 | +100g |

#### 6. 每日限制

```
每日参与次数: 3次
每日重置: 每日凌晨6点
超出限制: 提示"今日次数已用完"
```

## Formulas

### 1. 擀皮评分计算

```
# 擀皮完成度计算
size_ratio = actual_size / target_size  # 目标大小的比例
roundness = calculate_circularity(dough)  # 圆形度 (0-1)
thickness_uniformity = calculate_uniformity(thickness_map)  # 厚度均匀度 (0-1)

# 大小判定
size_score = {
    if 0.6 <= size_ratio <= 0.8: 1.0   # 完美
    elif 0.5 <= size_ratio < 0.6: 0.7   # 偏小
    elif 0.8 < size_ratio <= 0.9: 0.7   # 偏大
    else: 0.3                            # 太小或太大
}

# 形状判定
shape_score = roundness >= 0.8 ? 1.0 : roundness

# 厚度判定
thickness_score = thickness_uniformity >= 0.7 ? 1.0 : thickness_uniformity

# 擀皮基础分
rolling_base = size_score * SIZE_WEIGHT + shape_score * SHAPE_WEIGHT + thickness_score * THICKNESS_WEIGHT
rolling_points = rolling_base * ROLLING_MAX_POINTS  # 30分满分

# 等级判定
if size_score == 1.0 and shape_score >= 0.8 and thickness_score >= 0.7:
    rolling_grade = "perfect"
elif [size_score, shape_score, thickness_score].count(1.0) >= 2:
    rolling_grade = "good"
elif [size_score, shape_score, thickness_score].count(1.0) >= 1:
    rolling_grade = "normal"
else:
    rolling_grade = "fail"
```

### 2. 包馅评分计算

```
# 位置偏移计算
center_offset = distance(馅料中心, 面皮中心) / 面皮半径
position_accuracy = 1.0 - center_offset

# 包馅基础分
filling_points = min(position_accuracy / OPTIMAL_THRESHOLD, 1.0) * FILLING_MAX_POINTS  # 20分满分

# 等级判定
if center_offset <= 0.1:
    filling_grade = "perfect"
elif center_offset <= 0.2:
    filling_grade = "good"
elif center_offset <= 0.3:
    filling_grade = "normal"
else:
    filling_grade = "overflow"
```

### 3. 捏合评分计算

```
# 褶皱数量和位置
correct_pleats = count_correct_position_clicks()
total_pleats = count_total_clicks()
position_accuracy = correct_pleats / total_pleats

# 速度因子
avg_click_interval = total_time / total_pleats
if avg_click_interval <= OPTIMAL_INTERVAL:
    speed_factor = 1.0
else:
    speed_factor = OPTIMAL_INTERVAL / avg_click_interval

# 捏合基础分
pleating_base = position_accuracy * POSITION_WEIGHT + speed_factor * SPEED_WEIGHT
pleating_points = pleating_base * PLEATING_MAX_POINTS  # 50分满分

# 等级判定
if total_pleats >= 12 and position_accuracy >= 0.9:
    pleating_grade = "perfect"
elif total_pleats >= 10 and position_accuracy >= 0.8:
    pleating_grade = "good"
elif total_pleats >= 8:
    pleating_grade = "normal"
else:
    pleating_grade = "rough"
```

### 4. 单个饺子评分

```
# 等级到分数映射
grade_to_points = {
    "perfect": 30,
    "good": 20,
    "normal": 10,
    "fail": 0
}

# 单个饺子总分
rolling_points = grade_to_points[rolling_grade]
filling_points = grade_to_points[filling_grade]
pleating_points = grade_to_points[pleating_grade]

dumpling_score = rolling_points + filling_points + pleating_points

# 饺子等级判定
if dumpling_score >= 90: dumpling_rank = "完美"      # ⭐⭐⭐
elif dumpling_score >= 75: dumpling_rank = "优秀"     # ⭐⭐
elif dumpling_score >= 50: dumpling_rank = "合格"     # ⭐
else: dumpling_rank = "不合格"                        # 破皮
```

### 5. 连击计算

```
# 连击更新
if dumpling_score >= 50:  # 合格及以上
    current_combo += 1
    max_combo = max(max_combo, current_combo)
else:
    current_combo = 0

# 连击奖励
combo_bonus = 0
if current_combo >= 10: combo_bonus = 150
elif current_combo >= 8: combo_bonus = 100
elif current_combo >= 5: combo_bonus = 50
elif current_combo >= 3: combo_bonus = 20

# 总分累加
total_score += dumpling_score + combo_bonus
```

### 6. 最终得分计算

```
# 基础分
base_score = sum(dumpling_score for each_dumpling)

# 连击奖励
combo_bonus_total = sum(combo_bonuses)

# 总分
total_score = base_score + combo_bonus_total
```

### 7. 奖励计算

```
# 称号判定
if total_score >= 2500:
    title = "面点大师"
    base_coins = 500
elif total_score >= 2000:
    title = "包饺高手"
    base_coins = 300
elif total_score >= 1500:
    title = "包饺新秀"
    base_coins = 150
else:
    title = "参与奖"
    base_coins = 50

# 额外奖励
extra_coins = 0
if perfect_dumplings >= 5:
    extra_coins += 100
if is_new_record:
    extra_coins += 100

total_coins = base_coins + extra_coins
```

## Edge Cases

### 1. 擀皮边界

| 情况 | 处理方式 |
|------|----------|
| **面皮擀得太大** | 超出目标面积，显示警告，但允许继续 |
| **面皮擀得太小** | 低于最小面积，无法进入下一阶段 |
| **面皮形状不圆** | 圆形度计算，低于阈值则扣分 |
| **面皮厚薄不均** | 均匀度计算，显示提示 |
| **超时未完成** | 自动提交当前状态，根据完成度评分 |

### 2. 包馅边界

| 情况 | 处理方式 |
|------|----------|
| **馅料放偏** | 根据偏移程度扣分，最多偏移30% |
| **馅料太少** | 不扣分，但影响整体评分 |
| **馅料溢出** | 评为"溢出"，只能得5分 |
| **不放馅料** | 不允许跳过，必须放馅料 |
| **超时未放** | 自动放置在当前中心位置 |

### 3. 捏合边界

| 情况 | 处理方式 |
|------|----------|
| **褶皱位置错误** | 不计入正确褶皱，影响评分 |
| **点击速度太快** | 忽略间隔<0.1秒的点击 |
| **点击速度太慢** | 不影响，但影响速度分 |
| **只捏一半** | 进入下一阶段，按实际褶皱数评分 |
| **超时未完成** | 自动完成当前状态 |

### 4. 时间边界

| 情况 | 处理方式 |
|------|----------|
| **剩余时间<5秒** | 允许开始一个饺子 |
| **最后一个饺子未完成** | 自动提交，按完成度评分 |
| **游戏进行中切后台** | 暂停游戏 |
| **游戏被强制结束** | 不保存当前进度 |

### 5. 连击边界

| 情况 | 处理方式 |
|------|----------|
| **连击中断后恢复** | 从0开始重新计数 |
| **连续多个不合格** | 每次都重置为0 |
| **10连后继续连中** | 保持10连，不再额外加分 |
| **游戏结束时连中未结算** | 结算时计算最终连中数 |

### 6. 评分边界

| 情况 | 处理方式 |
|------|----------|
| **分数为负数** | 不可能，最少为0 |
| **分数超过理论上限** | 无上限，可无限得分 |
| **完美饺子计算** | 只计算最终饺子，不计算中间阶段 |
| **分数段临界** | ≥2500得"面点大师"，严格按≥判断 |

### 7. 奖励边界

| 情况 | 处理方式 |
|------|----------|
| **首次获得"面点大师"** | 只触发一次成就 |
| **连续刷新记录** | 每次都奖励 |
| **背包已满** | 奖励改为金币补偿 |
| **金币溢出** | 使用int64存储 |
| **完美饺子刚好5个** | 触发额外奖励 |

### 8. 每日限制边界

| 情况 | 处理方式 |
|------|----------|
| **今日次数已用完** | 按钮置灰，显示"今日次数已用完" |
| **跨日重置** | 每日凌晨6点重置 |
| **日期切换时参与** | 按进入时的日期计算次数 |

### 9. 设备/输入边界

| 情况 | 处理方式 |
|------|----------|
| **触屏设备** | 触摸滚动/点击捏褶 |
| **键盘设备** | 空格滚动/空格/方向键捏褶 |
| **手柄设备** | 扳机键滚动/A按钮捏褶 |
| **输入设备切换** | 立即生效，无需暂停 |

### 10. 性能/技术边界

| 情况 | 处理方式 |
|------|----------|
| **帧率过低** | 保持判定精度，简化动画 |
| **存档损坏** | 重置当日次数，使用默认值 |
| **网络断开** | 本地计时，成绩本地保存 |
| **版本更新** | 重新计算历史最高分（可选） |

### 11. 操作异常

| 情况 | 处理方式 |
|------|----------|
| **快速连续操作** | 每阶段有冷却时间 |
| **操作顺序错误** | 忽略错误操作，等待正确输入 |
| **同时按多个键** | 只响应第一个 |
| **鼠标/触屏漂移** | 使用平滑处理 |

### 参数调优建议

**让游戏更难**:
1. 减小 `GAME_DURATION` 至 45秒
2. 增大 `MIN_PLEATS` 至 10
3. 减小 `OPTIMAL_INTERVAL` 至 0.2秒
4. 增大 `MIN_ROUNDNESS` 至 0.9

**让游戏更简单**:
1. 增大 `GAME_DURATION` 至 90秒
2. 减小 `MIN_PLEATS` 至 6
3. 增大 `OPTIMAL_INTERVAL` 至 0.5秒
4. 减小 `MIN_ROUNDNESS` 至 0.7

**让游戏更有节奏感**:
1. 开启 `DIFFICULTY_RAMP`
2. 减小 `RAMP_INTERVAL` 至 每3个饺子
3. 减小 `DIFFICULTY_INCREASE` 至 3%
