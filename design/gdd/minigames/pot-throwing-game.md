## Overview

投壶小游戏是中秋节节日活动中的传统技巧玩法。玩家通过"蓄力+瞄准+时机"三要素将箭矢投入壶中。

游戏采用混合操作模式：
- **蓄力**：按住蓄力条控制投掷力度
- **瞄准**：调整投掷角度（上下左右微调）
- **时机**：壶口会周期性张开，需在最佳时机投掷

每场游戏有10支箭矢，根据投入数量、连中次数、完美投掷获得积分。

小游戏独立于主游戏运行，完成后根据成绩与活动系统交互进行奖励结算。

## Player Fantasy

投壶小游戏给玩家带来**传统技艺的韵味和精准控制的成就感**。玩家应该感受到：

- **蓄力的期待** — 按住蓄力条，感受力量的积累和释放的冲动
- **瞄准的专注** — 调整角度，寻找最佳抛物线的专注
- **时机的紧张** — 壶口张开的瞬间，心跳加速
- **投中的满足** — 箭矢入壶的清脆声响，带来满足感
- **连中的狂喜** — 连续投中多支的成就感
- **中秋的氛围** — 月光下的投壶场地，赏月品茗的传统节庆感

参考游戏：
- **Angry Birds**: 蓄力+弹道的经典玩法
- **投壶游戏**: 传统技巧游戏
- **高尔夫推杆**: 力度控制的精准感

**情感曲线**:
1. **开场**: 月光下的投壶场地，营造中秋氛围
2. **前期**: 熟悉蓄力和瞄准，手感逐渐建立
3. **中期**: 掌握时机节奏，开始尝试连中
4. **后期**: 追求完美投掷，挑战连中极限
5. **结算**: 查看成绩和奖励，分享喜悦

## Detailed Design

### Core Rules

#### 1. 操作三要素

##### 1.1 蓄力系统

```
蓄力操作: 按住蓄力键（空格/点击）
蓄力条: 0% → 100% → 循环

最佳蓄力区间: 70% - 90%
- 70-90%: 最佳力度，命中率最高
- < 30%: 力度不足，命中率极低
- > 95%: 力度过大，容易越过壶
```

##### 1.2 瞄准系统

```
瞄准操作: 方向键 / 摇杆
角度范围: -15° 到 +15° (左右微调)

瞄准辅助: 显示预测轨迹线
- 轨迹线根据蓄力值和角度计算
- 帮助玩家判断落点
```

##### 1.3 时机系统

```
壶口状态:
- 闭合状态: 2秒周期
- 张开状态: 持续0.5秒

时机判定:
- 壶口张开时投掷: 命中率+30%
- 壶口闭合时投掷: 正常命中率

视觉提示:
- 壶口张开前0.3秒闪烁提示
- 张开时壶口发光
```

#### 2. 投掷流程

```
┌─────────────────────────────────────────┐
│  1. 准备阶段 (3秒)                      │
│     └─ 显示规则，投壶演示                │
├─────────────────────────────────────────┤
│  2. 投掷阶段 (共10支箭)                 │
│     └─ 蓄力 → 瞄准 → 时机 → 投掷        │
│           ↓                             │
│     [蓄力条上升]                        │
│           ↓                             │
│     [调整角度]                          │
│           ↓                             │
│     [等待时机]                          │
│           ↓                             │
│     [松手投掷]                          │
│           ↓                             │
│     [判定结果]                          │
│           ↓                             │
│     [显示分数]                          │
├─────────────────────────────────────────┤
│  3. 结算阶段                            │
│     └─ 显示成绩 → 发放奖励              │
└─────────────────────────────────────────┘
```

#### 3. 命中判定

##### 3.1 命中计算

```
基础命中率 = f(蓄力值, 角度)

蓄力影响:
  < 30%: 命中率 × 0.3
  30-70%: 命中率 × (0.5 + 0.5×ratio)
  70-90%: 命中率 × 1.0  (最佳)
  90-100%: 命中率 × 0.8
  > 100%: 命中率 × 0.5

时机加成:
  张开时投掷: 命中率 += 30%
  闭合时投掷: 无加成

角度影响:
  偏离中心: 命中率 × (1 - 偏离度)
```

##### 3.2 判定结果

| 判定 | 条件 | 基础分 |
|------|------|--------|
| **完美投中** | 力度70-90% + 张开时 + 角度<3° | 200分 |
| **精准投中** | 力度40-95% + 角度<8° | 150分 |
| **普通投中** | 命中但偏离较大 | 100分 |
| **未投中** | 箭矢落地/弹开/越过壶 | 0分 |

#### 4. 连中系统

```
连中计数: 连续投中时连击数+1
连击中断: 投偏或未投中时重置为0

连中奖励:
  3连中: 额外+30分
  5连中: 额外+80分
  8连中: 额外+150分
  10连中(满中): 额外+300分
```

#### 5. 奖励结算

##### 5.1 分数段奖励

| 分数段 | 称号 | 金币奖励 |
|--------|------|----------|

| ≥1400分 | 投壶高手 | 300g |

##### 5.2 额外奖励

```

## Formulas

### 1. 蓄力值计算

```
# 蓄力条循环 0% → 100% → 0%
charging_value = (current_time % CHARGE_CYCLE) / CHARGE_CYCLE
CHARGE_CYCLE = 2.0  # 2秒一个完整周期

# 蓄力值映射到力度
power = min(charging_value * 1.2, 1.0)  # 超出100%后力度锁定
```

### 2. 命中率计算

```
# 基础命中率
base_hit_rate = 0.6  # 60%

# 蓄力影响因子
power_factor = {
    if power < 0.3: 0.3
    elif power < 0.7: 0.5 + power * 0.5
    elif power <= 0.9: 1.0
    elif power <= 1.0: 1.0 - (power - 0.9) * 4
    else: 0.5
}

# 角度影响因子
angle_factor = 1.0 - abs(angle) / MAX_ANGLE
MAX_ANGLE = 15  # 度

# 时机加成
timing_bonus = is_pot_open ? 0.3 : 0.0

# 最终命中率
hit_rate = clamp(base_hit_rate * power_factor * angle_factor + timing_bonus, 0.0, 1.0)
```

### 3. 命中判定

```
# 随机判定是否命中
is_hit = random() < hit_rate

if is_hit:
    # 根据角度、时机和力度判定等级
    # 完美: 力度70-90% + 壶口张开 + 角度<3°
    if abs(angle) <= 3 and is_pot_open and power >= 0.7 and power <= 0.9:
        result = "perfect"   # 完美
    # 精准: 力度40-95% + 角度<8° (不要求壶口张开)
    elif abs(angle) <= 8 and power >= 0.4 and power <= 0.95:
        result = "precise"   # 精准
    else:
        result = "normal"     # 普通
else:
    result = "miss"          # 未投中
```

**注意**: 蓄力值 >95% 或 <40% 只能获得"普通"评价，即使命中。

### 4. 得分计算

```
# 基础分
base_score = {
    "perfect": 200,
    "precise": 150,
    "normal": 100,
    "miss": 0
}[result]

# 连中奖励
streak_bonus = 0
if result != "miss":
    if streak >= 10: streak_bonus = 300
    elif streak >= 8: streak_bonus = 150
    elif streak >= 5: streak_bonus = 80
    elif streak >= 3: streak_bonus = 30

# 总分
total_score = base_score + streak_bonus
```

### 5. 连中计算

```
# 连中更新
if result != "miss":
    current_streak += 1
    max_streak = max(max_streak, current_streak)
else:
    current_streak = 0

# 连中状态
streak_level = {
    if current_streak >= 10: "legendary"   # 传说
    elif current_streak >= 8: "master"      # 大师
    elif current_streak >= 5: "expert"      # 高手
    elif current_streak >= 3: "good"         # 不错
    else: "normal"
}
```

### 6. 奖励计算

```

if total_score >= 1800:
    title = "投壶圣手"
    base_coins = 500
elif total_score >= 1400:
    title = "投壶高手"
    base_coins = 300
elif total_score >= 1000:
    title = "投壶新秀"

## Edge Cases

### 1. 蓄力边界

| 情况 | 处理方式 |
|------|----------|
| **按住蓄力键不放** | 蓄力条循环，不会卡住 |
| **蓄力值过低(<10%)** | 命中率大幅降低，显示"力度不足" |
| **蓄力值过高(>100%)** | 超过100%后力度锁定在100%，不继续增加 |
| **快速点按** | 视为短蓄力，力度很低 |

### 2. 瞄准边界

| 情况 | 处理方式 |
|------|----------|
| **角度超过限制** | 限制在±15°，不超出 |
| **瞄准时松手** | 使用当前角度投掷 |
| **不调整角度** | 使用默认0°（正中） |

### 3. 时机边界

| 情况 | 处理方式 |
|------|----------|
| **壶口张开瞬间投掷** | 算作张开时投掷 |
| **壶口闭合瞬间投掷** | 算作闭合时投掷 |
| **箭矢飞行中壶口状态变化** | 以投掷瞬间的壶口状态为准 |

### 4. 投掷边界

| 情况 | 处理方式 |
|------|----------|
| **飞行中切后台** | 暂停，等待恢复 |
| **同时按多个方向键** | 只计算最后一个 |
| **投掷后立即下一支** | 需等待当前箭矢落地判定完成 |
| **快速连续投掷** | 忽略，必须等待当前投掷完成 |

### 5. 命中判定边界

| 情况 | 处理方式 |
|------|----------|
| **刚好擦边** | 根据是否进入壶口范围判定 |
| **弹入壶中** | 不算投中，箭矢弹开 |
| **完美临界(角度3.1°)** | 算作精准，不算完美 |
| **多次碰撞** | 以第一次落点为准 |

### 6. 连中边界

| 情况 | 处理方式 |
|------|----------|
| **10支全中** | 触发特殊奖励+动画 |
| **连中中断** | 连击重置为0，连中奖励不累计 |
| **10支全中但有失误后恢复** | 不算全中 |

### 7. 积分边界

| 情况 | 处理方式 |
|------|----------|
| **积分超过理论上限** | 无上限，可无限得分 |
| **历史最高分更新** | 对比本地存储的最高分 |
| **分数段临界** | ≥1800得"投壶圣手"，严格按≥判断 |

### 8. 奖励边界

| 情况 | 处理方式 |
|------|----------|
| **首次获得"投壶圣手"** | 只触发一次成就 |
| **连续刷新记录** | 每次都奖励 |
| **背包已满** | 奖励改为金币补偿 |
| **金币溢出** | 使用int64存储 |

### 9. 每日限制边界

| 情况 | 处理方式 |
|------|----------|
| **今日次数已用完** | 按钮置灰，显示"今日次数已用完" |
| **跨日重置** | 每日凌晨6点重置 |
| **日期切换时参与** | 按进入时的日期计算次数 |

### 10. 设备/输入边界

| 情况 | 处理方式 |
|------|----------|
| **触屏设备** | 点击蓄力，滑动瞄准 |
| **键盘设备** | 空格蓄力，方向键瞄准 |
| **手柄设备** | RB蓄力，摇杆瞄准 |
| **输入设备切换** | 立即生效，无需暂停 |

### 11. 网络/性能边界

| 情况 | 处理方式 |
|------|----------|
| **网络断开** | 本地计时，成绩本地保存 |
| **帧率过低** | 保持物理判定精度，简化动画 |
| **崩溃恢复** | 不支持中途恢复，需重新开始 |

### 12. 投壶物理边界

| 情况 | 处理方式 |
|------|----------|
| **箭矢飞行轨迹** | 使用抛物线模拟 |
| **风力影响** | 可选，默认无风 |
| **重力设置** | 固定值，确保手感一致 |

### 参数调优建议

**让游戏更难**:
1. 减小 `POT_OPEN_DURATION` 至 0.3秒
2. 减小 `POT_OPEN_BONUS` 至 0.2
3. 增大 `MAX_ANGLE` 至 20°
4. 减小 `ARROW_COUNT` 至 5支

**让游戏更简单**:
1. 增大 `POT_OPEN_DURATION` 至 1.0秒
2. 增大 `POT_OPEN_BONUS` 至 0.5
3. 减小 `MAX_ANGLE` 至 10°
4. 增大 `BASE_HIT_RATE` 至 0.7

**让游戏更有挑战性(高手向)**:
1. 减小 `POT_OPEN_INTERVAL` 至 1.5秒
2. 减小 `POT_WARN_BEFORE` 至 0.1秒
3. 减小连中奖励，鼓励单次精准而非连中
