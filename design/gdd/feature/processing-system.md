## Overview

加工系统是游戏的核心生产系统之一，包含21种加工机器和150+加工配方。玩家通过在家中工坊放置加工机器，将原材料（农作物、矿石、动物产品等）转化为高价值的加工品（如酒、奶酪、织物、工具材料等）。系统与畜牧、农场、钓鱼、采矿等多个生产系统联动，是资源转化链的重要环节。

## Player Fantasy

加工系统给玩家带来**创造的满足感和规模化的成就感**。玩家应该感受到：

- **原料的神奇转化** — 看着水果变成美酒，矿石变成金属锭
- **规模生产的喜悦** — 多台机器同时运转，产出源源不断
- **副产品的惊喜** — 意外产出稀有材料（如育种种子、梦丝）
- **自动化的高效** — 自动收集功能让玩家无需频繁操作

**Reference games**: Stardew Valley 的加工简单直接；Factorio 的流水线更有深度。

### States and Transitions

#### 机器槽位状态

| 状态 | 描述 | 条件 |
|------|------|------|
| **Idle** | 空闲 | 机器已放置，无配方 |
| **Processing** | 加工中 | 正在加工，倒计时中 |
| **Ready** | 可收取 | 加工完成，等待收取 |

**状态转换图**:
```
Idle → Processing: 投入原料
Processing → Idle: 取消加工（退回原料）
Processing → Ready: 加工完成
Ready → Idle: 收取产物
Ready → Processing: 自动续产（虚空箱模式）
```

#### 机器放置/拆除

| 状态 | 描述 | 操作 |
|------|------|------|
| **Placed** | 已放置 | 工坊中可见 |
| **Removed** | 已拆除 | 材料返还，移出工坊 |

**拆除返还**:
- 制作材料 100% 返还
- 已完成产物收取
- 进行中原料退回

## Formulas

### 1. 加工时间公式

```
# 基础加工时间（天）
base_processing_days = recipe.processing_days

# 仙缘能力「织速」减免
# ⚠️ 临时 ID: gui_nv_1 - 待 P07 HiddenNPCSystem 设计后更新
if has_hidden_npc_ability('gui_nv_1') and machine_type == 'loom':
    final_days = max(1, ceil(base_days * 0.7))
else:
    final_days = base_days

# 加工时间范围：1-7天
final_days = clamp(final_days, 1, 7)
```

### 2. 产物品质公式

```
# 产物品质 = 投入原料最低品质
output_quality = min(all_input_ingredients.quality)

# 例外：部分机器可能有品质提升（未来可扩展）
# 暂定：产出品质 = 投入品质
```

### 3. 种子制造机公式

```
# 基础产出：每种作物产出2颗种子
output_quantity = 2

# 育种种子概率
breed_seed_chance = 0.05

# 触发条件
if machine_type == 'seed_maker':
    if random() < breed_seed_chance:
        add_item('genetic_seed', 1)  # 育种种子
        add_log('种子制造机额外产出了一颗育种种子！')
```

### 4. 织布机公式（仙缘能力）

```
# 「梦织」能力：8%概率额外产出梦丝
```

## Edge Cases

### 1. 机器放置边界情况

| 情况 | 处理方式 |
|------|----------|
| 工坊已满 | 无法放置，提示机器上限 |
| 材料不足 | 无法制造，提示缺少材料 |
| 机器已存在 | 不限制，可放置多台同类型 |

### 2. 加工边界情况

| 情况 | 处理方式 |
|------|----------|
| 原料不足 | 无法开始加工 |
| 加工中取消 | 退回 100% 原料 |
| 加工完成不收取 | 保持 Ready 状态 |
| 品质指定无效 | 使用最低可用品质 |

### 3. 自动续产边界情况

| 情况 | 处理方式 |
|------|----------|
| 虚空原料箱无料 | 回到 Idle 状态 |
| 虚空成品箱满 | 放入玩家背包 |
| 背包也满 | 丢弃产物，提示警告 |
| 无虚空箱配置 | 需手动收取 |

### 4. 拆除机器边界情况

| 情况 | 处理方式 |
|------|----------|
| 加工中拆除 | 退回原料，收取产物（如有） |
| 完成后拆除 | 收取产物 + 返还材料 |
| 空闲时拆除 | 返还 100% 材料 |

### 5. 品质边界情况

| 情况 | 处理方式 |
|------|----------|
| 多种品质原料 | 取最低品质作为产出品质 |
| 无指定品质 | 自动选择最低品质 |
| 指定高品质无货 | 回退到可用最低品质 |

### 6. 多日不上线处理

当玩家多日不上线时：
```
1. 加工进度: 上线时计算所有机器的累计天数
2. 完成的加工: 保持 Ready 状态或自动收取（如果有虚空箱）
3. 自动机器（蜂箱/蚯蚓箱）: 计算累计产出
4. 机器状态: 保持不变
```
